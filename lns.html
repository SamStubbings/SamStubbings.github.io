Life Not Supported

This project was made for my dissertation looking into how terrain features affect the gameplay of a Multiplayer Team-Based shooter. Also, game recording were taken using event data within the game. This allowed game to be re-ran inside the unity editor for metric to be generated.

How it was made

Perlin noise hills. clustered tree approach. dithered bands for textuing. Forge networking to handle basic networking code.
