2021 - Game made for Steam Release

Life Not Supported

Available on Steam

Unity / PCG

A Singleplayer Space Survival-Simulator game with basebuilding and crafting elements. Made as an "indie" developer on the Steam storefront.

2020 - Solo (3 Month Project)

Voxel Dungeon Crawler

Unity / PCG / AI

Implementing and improving upon Pittmans voxel dungeon generation algorithm to create an FPS roguelike tech demo.
Also has a working level editor, save system, runtime voxel destruction, biomes, and texture-pack support.

2019 - Dissertation Artifact

Multiplayer Procedural Team-Based FPS

Unity / Networking / PCG

2-10 player FPS with lobby servers.
Octaved perlin noise for hills, perlin-based tree clusters, and dithered height-based texturing.
Also a working match replay and metric gathering system.

2020/2021 - Team Project

Vestigiascope

Lead Developer

Available on the AppStore & PlayStore (1000+ Downloads)

Unity / AR / iOS / Android

Implementing given design with a fixed timeframe for a public event using phone based Augmented Reality.
